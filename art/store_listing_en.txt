Love to read? Having trouble following your best authors on samlib.ru?

SI Tracker allows you to stay informed of any updates your authors post. 
Use it comfortably on devices of any size with responsive buttery UI.

***Intended for use with http://samlib.ru and http://budclub.ru only

Planed features:

• Publication description with ratings.
• Updated publications in a separate tab
• Publication details
• samlib user profile
• View publication related images
• Reading, posting and tracking comments.

If you encounter any issue, please feel free to report it.


What's new:

• Added import and export support. txt and xml formats are supported.
• Added google back-up. Now the content is synchronized between your devices. Not guaranteed, "backup" and "restore" must be enabled in device settings.
• Added possibility to add authors from budclub.ru
• Support for interactive download folder choosing.
• Fixed author removal. Now all related publications are removed as well.
